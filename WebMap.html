"A Look at Food"

This interactive map provides census block data on households eligible for SNAP benefits, based on American Community Survey 5-Year data (2016-2020). This is the starting point in my journey to learn the ArcGIS API for Javascript, expand my Web and Map Application building skills and strengthen my Spatial Data Analysis experience. The long-term project goal will be a comprehensive food access web map using ESRI capabilities as well as other open source resources, data and APIs. Stay tuned!!
